# Mobile Panchangam Image Share Design Specification

## Overview
This document specifies the design for allowing users on mobile (and desktop) devices to share the complete Daily Panchangam details as a single, beautifully formatted, high-resolution PNG image directly to WhatsApp, Telegram, or native mobile share targets.

## User Intent & Requirements
- **Primary Goal**: Provide a single-tap "Share as Image" feature on `PanchangamPage`.
- **Image Content**: Include the full detailed Panchangam details (Date, Location, Sun/Moon rise/set, Thithi, Nakshatra, Yogam, Rashi, Chandrastamam, Subha Muhurtham, Vasthu Neram, Nalla Neram, Gowri Nalla Neram, Rahu/Yama/Gulika Kalam, and the complete 24 Horai table).
- **Branding Footer**: Include `Generated by https://tinyurl.com/drik-vedic` at the bottom of the exported card.
- **Mobile Integration**: Trigger `navigator.share({ files: [pngBlob] })` on mobile browsers (WhatsApp, Telegram, etc.) with direct PNG download fallback on desktop.

---

## Technical Approach & Architecture

### Client-Side Render & Capture Architecture
1. **Dependency**: Add `html2canvas` to `frontend/package.json`.
2. **Hidden Export Card Component (`PanchangamShareCard`)**:
   - A fixed-width (`1080px`) off-screen React container rendered during share generation.
   - Styled with a dark astronomical theme (`#13141c` background, `#ffd700` gold accents, clean cards, and high contrast text).
   - Contains all localized strings based on `settings.language` (Tamil, English, Hindi, etc.).
3. **Capture & Share Workflow**:
   - User clicks **"đź“˛ Share as Image"** button on `PanchangamPage`.
   - Component temporarily mounts `<PanchangamShareCard>` off-screen (`position: absolute; left: -9999px; width: 1080px`).
   - `html2canvas` converts the DOM tree into an HTML5 `<canvas>` element.
   - Canvas calls `toBlob('image/png', 1.0)`.
   - If `navigator.canShare({ files })` is supported (mobile browsers): invoke `navigator.share({ files: [file], title: 'Daily Panchangam' })`.
   - If `navigator.share` is unavailable: trigger direct file download `Panchangam_<date>.png`.
