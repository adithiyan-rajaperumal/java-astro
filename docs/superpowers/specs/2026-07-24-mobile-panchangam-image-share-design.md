# Mobile Panchangam Image Share Specification (v3 - Compact & Official Logo)

## Overview
This document specifies the updated design for sharing the Daily Panchangam details as a compact PNG image card featuring the official `<AstroLogo>` component and an optimized height layout (~45% shorter).

## User Intent & Requirements
- **Official App Logo**: Replace generic emoji header with the official `<AstroLogo>` SVG component.
- **Height & Whitespace Optimization**: Reduce card height significantly by tightening margins, compacting padding, and rendering the 24 Horai table as a **6-column × 4-row grid** (`gridTemplateColumns: 'repeat(6, 1fr)'`).
- **Complete Localized Content**: Maintain 100% of all localized Panchangam details (Date, Location, Sun/Moon times, Thithi, Nakshatra, Yogam, Karanam, Rashi, Chandrastamam, Subha Muhurtham, Vasthu, Nalla Neram, Gowri Nalla Neram, Nakshatra Yogam, Abhijit Muhurtham, Rahu/Yama/Gulika Kalam, 24 Horais, and `Generated by https://tinyurl.com/drik-vedic` footer).

## Technical Approach & Architecture

### `PanchangamShareCard` Component Updates
- Import `AstroLogo` from `../components/AstroLogo`.
- Set container width: `1080px`, padding: `18px`, gap: `10px`.
- 24 Horais grid: `display: grid; gridTemplateColumns: repeat(6, 1fr); gap: 6px`.
